Today we shipped Acme v1, an Eleventy-powered rebuild of the original
fixture site. The new build runs `eleventy` and emits ACT-protocol
files alongside the static export.

## Highlights

The build now ships PRD-408 ACT artifacts on every deploy and passes
the validator at the Core conformance band.
