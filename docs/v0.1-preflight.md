## 4. Hosted validator (Q8 Option 3 — GitHub Pages SPA)

`apps/validator-web/` landed at `53ba703` immediately after this
pre-flight commit. Vite + vanilla TS SPA, browser-native ES2022,
208 KB / 62 KB gzip total. Wraps `@act-spec/validator` for browser
consumption (no second parser — `@act-spec/validator` ships
`compileSchemasFromRaw` + `setCompiledSchemas` exports so the SPA
seeds the compiled-schema cache from a Vite-bundled raw set without
calling the Node-only `loadSchemas`).

CORS UX surfaced per Q8: top-of-page `.cors-notice` banner; on URL-
fetch failure, a `.cors-warning` with a "switch to paste mode"
button auto-toggles the input mode.

Tests: 23 SPA unit tests PASS, 6 SPA conformance checks PASS, full
`pnpm -r conformance` 0 failures, validator unchanged at 214 PASS.

Pages deploy: workflow at `.github/workflows/pages.yml` is wired and
trigger-ready (push-to-master + workflow_dispatch). **Held until
v0.2** — no public Pages deploy in v0.1.

**Verdict:** GREEN.
